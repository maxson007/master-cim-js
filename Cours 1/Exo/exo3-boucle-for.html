Cours 1 Javascript - Exercice 3 Boucle for

Calculez et affichez la factorielle de n où la valeur de n est donnée par l’utilisateur : n! = 1*2*…*n


Exemple :
  • 1! = 1
  • 2! = 1*2 = 2
  • 3! = 1*2*3 = 6
let value = prompt("Entrez une valeur entre 1 et 100"); let n=parseInt(value); let result=1; for (let i=1;i<=n;i++){ result*=i; } alert(`Factorielle de ${value} est ${result}`);
